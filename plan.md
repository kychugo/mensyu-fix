### 5.2 學習地圖（`#learning-map`）

**概念**：以「地圖探索」呈現進度，每位古人是一條支線地圖，共 5 關。

```
[選擇古人] ─→ 蘇軾地圖: 關1 ✓ → 關2 ✓ → 關3 🔒 → 關4 🔒 → 關5 🔒
             韓愈地圖: 關1 ✓ → 關2 🔒 → ...
```

**各關解鎖條件**：
- 第 1 關：預設解鎖
- 第 N 關：第 N-1 關測驗通過（≥ 60 分）

**每關流程（五步學習法）**：

```
① 閱讀原文   → ② 翻譯理解   → ③ 關鍵詞測驗   → ④ 古人對話   → ⑤ 闖關測驗
  (原文展示)     (逐字翻譯)     (填空/配對)      (AI聊天)       (5道題 ≥60分通關)
```

#### ① 閱讀原文
- 顯示完整文言文，常見字詞**粗體**標示
- 右側（桌面）/ 上方摺疊（手機）顯示原文
- 字詞點擊→即時解釋彈出（來自 v3 `.character` 功能）

#### ② 翻譯理解
- 呼叫 AI 生成逐字翻譯（`請將以下文言文逐字翻譯...`）
- 翻譯結果分三欄：「原文」→「逐字對譯」→「句子意思」
- 翻譯快取至 localStorage（以 `authorId + levelId` 為 key）
- 「點擊顯示」遮蓋模式：學生可先嘗試自己翻譯

#### ③ 關鍵詞測驗（**新增小遊戲**）
- **文磚挑戰（Breakout）**：磚塊上有文言字詞，打中正確磚塊得分（來自 `mensyu-tran.html`）
- **配對遊戲（Matching）**：將文言詞語與白話解釋配對（來自 README 提及的配對遊戲）
- **填字遊戲（Fill-in）**：句子挖空，選出正確文言字詞

#### ④ 古人對話（**核心新功能**）
- 解鎖「與古人聊天」後，進入聊天室界面
- AI 扮演當前關卡的古人（蘇軾/韓愈），以角色個性回答
- 可詢問：本篇作品的背景、語句意思、人生哲學
- 系統提示詞（System Prompt）內嵌角色 personality + 篇章 context

#### ⑤ 闖關測驗
- AI 生成 5 道選擇題（篇章理解 + 字詞解釋 + 句式分析）
- ≥ 3 題（60 分）通過 → 解鎖下一關
- 失敗可重試，但須重新學習翻譯（ ② ）

## 六、API 整合方案

### 6.1 金鑰與模型設定

```javascript
const API_KEYS = [
  { sk: 'sk_I9LbeRaewORSMEdm2ontKkJEHgimbE1v', pk: 'pk_ZQ4XnvfBU2tu6riY' }
  // 未來可擴充更多金鑰
];
const MODELS = ['deepseek', 'glm', 'qwen-large', 'qwen-safety'];
const API_URL = 'https://gen.pollinations.ai/v1/chat/completions';
```

### 6.2 自動切換邏輯

```
呼叫 API
 ├─ 成功 → 返回結果
 └─ 失敗（4xx/5xx/timeout）
      └─ 切換至下一個 model（循環）
           └─ 若所有 model 失敗 → 顯示「AI 暫時不可用」提示
```

- 每次請求先嘗試當前 model
- 失敗後依序嘗試 `MODELS` 陣列中的其他 model
- 失敗狀態記錄在 `console.warn`，不影響用戶體驗
